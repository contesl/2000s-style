2000s Style: Personal Objects Collection
El proyecto es un sitio de eCommerce que implementa el concepto de "Moda Rapida": articulos de uso personal a precios asequibles.

Estructura del proyecto
Directorios
2000S-Style: directorio raiz de la aplicacion
\src : source files
.\components: componentes reutilizables en diferentes paginas
.\pages : paginas de la aplicacion. Basicamente implementan el menu
.\styles : paginas de estilo. La aplicacion utiliza styles.css
.\img : imagenes utilizadas en las paginas y componentes
